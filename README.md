## Core Features

- Built using **React.js** ‚Äî component-based architecture, easy to maintain and extend.  
- Styled with **Tailwind CSS** for utility-first, responsive and mobile-friendly design.  
- Real-time Color Preview ‚Äî Instantly see the selected color.
- HEX & RGB Conversion ‚Äî Automatically converts between both formats.
- One-click Copy ‚Äî Copy HEX or RGB value to clipboard easily.
- Responsive UI ‚Äî Fully optimized for mobile, tablet, and desktop.
- Fast & Lightweight ‚Äî Built with clean and modern React architecture.
- Color Input & Slider Support ‚Äî Multiple ways to choose your color.
